# Sartu PHP-Version

Die aktuelle Projektdokumentation steht in `README.md`.

Kurzfassung:

1. MySQL-Datenbank anlegen.
2. `database/mysql-schema.sql` importieren.
3. `includes/config.local.example.php` als `includes/config.local.php` kopieren und ausfüllen.
4. Ersten Admin in der Tabelle `profiles` anlegen.
5. Lokal optional mit `php -S localhost:8080 router.php` testen.
